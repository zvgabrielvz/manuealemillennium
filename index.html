7 • Substituição de estoque

  1. Logistica > Estoque > Inventário
  2. Incluir Inventário
    • Preencher nome inventário
    • Preencher filial
    • Preencher tipo de inventário
    • Efetivar
  3. Ações > Liberar contagem
  4. Logistica > Estoque > Contagem de Inventário
    • Incluir Contagem
    • Selecionar inventário
    • em Coletor de dados selecionar arquivo de dados
    • Escolher um item pelo EAN para verificar se houve alteração
    • Criar bloco de notas com o item no padrão do layout adicionando 1 em quantidade (precisa ter 5 dígitos no final)
    • Efetivar
    • Mudar para filtro iniciado e Selecionar inventário (iniciado)

    • Efetivar

Manuais Específicos

ETIQUETA DE VENDA (Verificar mais detalhes)

  1. Entrar nos pedidos de venda e buscar pelo pedido
  2. Pegar código cliente e pesquisar na movimentação de venda
  3. Ações -> etiqueta de transporte -> Sigep

INSTALAÇÃO SISTEMA

  1. Instalar normal, entrar no local de instalação
  2. Abrir wtgconfig, colocar IP, configurar 2 campos 99 (se caso não der certo, desmarcar forçar/outros campos
  3. Abrir gerenciador de usuários para carregar o restante, depois de tudo, abrir business millennium
  4. IP: 179.124.195.142

RESOLVENDO ERROS Allpick

  1. Sempre deixar tela de pedido de venda e pré faturamento abertas
  2. Quando for erro de cadastro
    • Abrir préfat, pesquisar pelo número do cliente no pedido de venda
    • Entrar no pedido e clicar em cima de cliente com o botão direito -> alterar cliente
  3. Caso o erro seja “não conferido”
    • Pesquisar pelo pré-faturamento
    • Ações -> conferir pré fat

EMBARQUES

  1. Conferir todos os dias no começo do dia os embarques
    • Logística -> Expedição -> Embarque
  2. Conferir se há embarques “em preparação”
    • Ações -> Altera Status
    • Digitar placa ABC0001 ou ABC0002 (depende da filial)
    • Alterar Status LIBERADO PARA EMBARQUE
    • Alterar Status EMBARQUE CONCLUÍDO
  3. Verificar se há pedidos para realizar embarques, pedidos que já estão embarcados no Allpick porém não foram para embarque
    • Incluir embarque, escolher filial
    • Escolher transportadora
    • Clicar em itens de embarque e clicar na lupa
    • Filtrar por data
    • Efetuar embarque dos pedidos que estão marcados como “embarcados” no Allpick

CADASTRO DE PRODUTO

  1. Utilizar um produto como exemplo
    • Ações -> Copiar
    • Mudar nome
    • Alterar código (sku)
    • Alterar referência (sku)
    • Olhar aba Vitrine e escolher categoria do produto, se não tiver, colocar “sem categoria”
    • Buscar produto para colocar EAN
  2. Ações -> código de barras -> preencher EAN
  3. Colocar preços
  4. Ações -> preço de -> preencher
  5. Ações -> Atacado -> preencher
    • Entrar na Vitrine para verificar se os produtos estão lá, senão estiver, clicar em óculos de sol, adicionar produtos e selecionar produtos
  6. Alterar preços
    • Vendas-> Formação de preço
  7. Preços de custo/venda
  8. Cadastrar ID Externo
    • Vitrine de produtos
    • Pesquisar produto (2cliques)
    • Aba s”SKU”, no último campo preencher ID externo
  9. Buscar produto na VETEX, copiar número entre parênteses “()”, esse é a ID externa do produto.

LANÇAMENTO DE XML

  1. Abrir Recebimento de compra
  2. Filial 00204 – Recebimento WMS
  3. Importa XML -> escolher arquivo
  4. Verificar item a item (principalmente quantidade) comparando com o arquivo romaneio excel e digitar o sku do produto
  5. Depois de certom dar OK e confirmar se a quantidade total de peças do Millennium está igual da planilha de romaneio
  6. Cond pagto 30 dias
  7. Efetivar
  8. Transp. Total
  9. Tipo frete
  10. Finalizar operação (f10)
  11. Próximo passo
  12. Conferir se a nota foi lançada corretamente no consulta movimentações de compra

CONFERIR LOG DE ERRO (todos os dias)

  1. Pesquisar no menu “LOG”
  2. Vitrine log de integração
  3. Filtrar por (produto, estoque, preço e pedido)
  4. Marcar flag de “erro” e “última atualização”
  5. Filtrar 2 dias para trás
  6. Exemplos de erro
    • Erro ao importar pedido -> Id do sku não encontrado
  7. Isso provavelmente acontece quando no pedido tem alguma bolsa ou acessório, pois não tem esses itens no Millennium
  8. Passar para o Alisson desmembrar o pedido, para a bolsa ir para o sistema “bling” e o óculos ficar sozinho para se comunicar com o sistema

COPIAR PEDIDO

  1. Ações -> Copiar pedido
  2. *Se caso a filial estiver 1, trocar para filial 2
  3. *Caso o pedido fique "AG ESTOQUE", perguntar para o alisson se tem estoque do produto
  4. Efetivar
  5. Depois, seguir manual do Criar pedido

ERRO DE PEDIDO E PRECISANDO COPIAR UM PEDIDO (pode ser um pedido com erro que precisa ser feito outro)

  1. Procurar na tela de vendas o pedido
  2. Suspender operação
    • Ações -> suspender operação
  3. Cancelar pré faturamento
    • Excluir expedição
  4. Ações -> expedição
